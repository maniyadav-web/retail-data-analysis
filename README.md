## Online Retail Data Analysis (EDA Project)

# 🧾 Project Description

This project presents an Exploratory Data Analysis (EDA) of the Online Retail dataset. The dataset contains transactional data from a online retail store. The goal is to analyze customer purchasing behavior, sales trends, and product performance to derive meaningful business insights.

The analysis helps in understanding revenue patterns, identifying top customers/products, and supporting data-driven decision-making for business growth.

# 🎯 Objectives
Understand customer purchasing behavior
Analyze monthly and seasonal sales trends
Identify top-selling products
Find top customers based on revenue
Detect missing values and data quality issues
Generate business insights from transactional data

# 📂 Dataset

The dataset used in this project is publicly available:

🔗 https://raw.githubusercontent.com/guipsamora/pandas_exercises/master/07_Visualization/Online_Retail/Online_Retail.csv

# 📊 Exploratory Data Analysis Performed
Data cleaning (handling missing values & duplicates)
Feature engineering (Revenue calculation)
Time-based analysis (monthly sales trends)
Top product analysis
Customer segmentation based on spending
Outlier detection in transaction data

# 🔍 Key Insights
Certain months show significantly higher sales (seasonal trends observed)
A small group of customers contributes to a large portion of revenue
Some products dominate overall sales performance
Data contains missing values and negative quantities (returns)
